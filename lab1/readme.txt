1. Команда cd pmat/. Открываем нужную папку
2. Команда mkdir lab1/. Создаём папку для лабораторной работы №1
3. Команда touch lab1/rand.py touch lab1/division.py touch lab1/sqrt.py. Создаём файлы
4. Команда vim lab1/rand.py vim lab1/division.py vim lab1/sqrt.py. Пишем код в файлах
5. Команда chmod +x lab1/rand.py lab1/division.py lab1/sqrt.py. Делаем файлы исполняемыми
6. Команда ./lab1/rand.py > output.txt; echo "10" | ./lab1/division.py >> output.txt; echo "16" | ./lab1/sqrt.py >> output.txt. Запускаем программы с перенаправлением вывода
7. Результат сохранён в файл output.txt
