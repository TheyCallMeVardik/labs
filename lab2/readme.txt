1. Команда cd pmat/. Открываем нужную папку
2. Команда mkdir lab2/. Создаём папку для лабораторной работы №2
3. Команда touch lab2/greet.py. Создаём файл
4. Команда vim lab2/greet.py. Пишем код в файле
5. Команда chmod +x lab2/greet.py. Делаем файл исполняемым
6. Команда ./lab2/greet.py > output.txt. Запускаем программу с перенаправлением вывода
7. Результат сохранён в файл output.txt
